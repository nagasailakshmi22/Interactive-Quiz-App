# Interactive Quiz Application

A dynamic full-stack quiz application built with Node.js, Express.js, MongoDB, and JavaScript.

##  Features
- Interactive quiz interface with real-time feedback
- RESTful backend APIs for quiz management
- Persistent data storage using MongoDB
- Dynamic question rendering on the frontend
- Score tracking and result display

##  Tech Stack
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Frontend:** HTML, CSS, JavaScript
- **API:** REST API design

##  How to Run
1. Clone the repository
   git clone https://github.com/nagasailakshmi22/Interactive-Quiz-App.git
2. Install dependencies
   npm install
3. Start MongoDB locally
4. Run the server
   node app.js
5. Open browser at http://localhost:3000

## Project Highlights
- Designed with API-first approach
- MongoDB used for storing questions and scores
- Scalable backend structure suitable for AI-powered extensions
